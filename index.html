Animations: What are they?

CSS Animations are made up of two basic building blocks:

  • Keyframes:
    • Name of the animation -- e.g. bounceIn, scale
    • Stages of the animation -- represented in %-ages
    • CSS Properties -- e.g. opacity, transform

Animations: What are they?

  • Animation Properties
    • Assign the @keyframes to the element you want animated
    • Define HOW it's animated

EXAMPLE

This animation has two stages: In the first stage (0%), the element is scaled at 100 percent using transform scale; the second stage (100%) is scaled at 130 percent. 

 @keyframes scale {
  0% {
    transform: scale(1);
     -webkit-transform: scale(-100px);
  }
  
  100% {
    transform: scale(1.3);
    -webkit-transform: scale(1.3);
  }
}

Transitions: What are they?

 

  • Allow you to control the speed of animations when changing properties.
  • Cause the changes in a property to take place over a period of time.

Transitions: What are they?

 

  • Four Transition Related Properties:
    • transition-property -- determines exactly what properties will be altered
    • transition-duration -- length of time for transition to take place
    • transition-timing-function -- sets speed at which a transition moves
    • transition-delay -- sets how long the transition should stall before execution

EXAMPLE

This transition eases the background-color out to white (from salmon); and the text color from white to grey with a :hover state over a 1 second period. 

 .square:hover {
  background-color: white;
  width: 150px;
  height: 50px;
  color: #666;
  text-transform: uppercase;
  -webkit-transition-property: background-color, color;
  transition-property: background-color, color;
  -webkit-transition-duration: 1s;
  transition-duration: 1s;
  -webkit-transition-timing-function: ease-out;
  transition-timing-function: ease-out;
}

Why are CSS Animations useful?

 

  • It's a powerful way to draw users attention to important areas of your product.
